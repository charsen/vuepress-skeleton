# Installation
```bash
# clone item
git clone git@github.com:charsen/vuepress-skeleton.git

# install dependencies
npm install

# serve with hot reload at localhost:6666
npm run dev

# build for production with minification
npm run build

# deploy to github page
npm run d

# build&&pm2
npm run server
